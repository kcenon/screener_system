## Acceptance Criteria

- [ ] `POST /v1/backtest` - 백테스트 실행 요청
- [ ] `GET /v1/backtest/{id}` - 백테스트 결과 조회
- [ ] `GET /v1/backtest/{id}/trades` - 거래 내역 조회
- [ ] `GET /v1/backtest/{id}/chart` - 수익률 차트 데이터
- [ ] `GET /v1/backtest/history` - 사용자 백테스트 히스토리
- [ ] Rate limiting: 10 req/hour (Free), 100 req/hour (Pro)
- [ ] OpenAPI 스펙 문서화
