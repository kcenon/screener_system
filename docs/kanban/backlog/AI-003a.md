## Description

차트 이미지 데이터셋 구축 및 CNN 모델 학습

## Acceptance Criteria

- [ ] 차트 이미지 생성 파이프라인 (OHLCV → 이미지)
- [ ] 10개 주요 패턴 라벨링 데이터셋 구축 (최소 1,000개/패턴)
- [ ] CNN 모델 학습 (ResNet/EfficientNet 기반)
- [ ] 패턴 인식 정확도 > 70%
- [ ] 모델 평가 및 검증

## Patterns to Detect

1. Head and Shoulders (머리어깨)
2. Double Top/Bottom (이중천장/이중바닥)
3. Triangle (삼각수렴)
4. Flag/Pennant (깃발/페넌트)
5. Cup and Handle (컵앤핸들)
6. Wedge (쐐기)
7. Channel (채널)
8. Rounding Bottom (원형바닥)
9. Rectangle (박스권)
10. Gap patterns (갭)
