## New Indicators

```python
# 투자자별 순매수
- foreign_net_buy_1d              # 외국인 당일 순매수
- foreign_net_buy_5d              # 외국인 5일 순매수
- foreign_net_buy_20d             # 외국인 20일 순매수
- institution_net_buy_1d          # 기관 당일 순매수
- institution_net_buy_5d          # 기관 5일 순매수
- institution_net_buy_20d         # 기관 20일 순매수
- individual_net_buy_1d           # 개인 당일 순매수

# 수급 비율
- foreign_ownership_pct           # 외국인 지분율
- foreign_ownership_change_1m     # 외국인 지분율 변화 (1개월)
- institution_ownership_pct       # 기관 지분율
- major_shareholder_pct           # 대주주 지분율

# 공매도
- short_balance                   # 공매도 잔고
- short_ratio                     # 공매도 비율
- short_balance_change_1w         # 공매도 잔고 변화 (1주)
- days_to_cover                   # 숏커버 소요일

# 신용/대차
- credit_balance                  # 신용잔고
- credit_ratio                    # 신용비율
- loan_balance                    # 대차잔고
```
