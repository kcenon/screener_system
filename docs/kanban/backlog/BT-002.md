## Description

백테스트 결과에 대한 종합적인 성과 지표 계산

## Acceptance Criteria

- [ ] 수익률 지표: 총수익률, CAGR, 월별/연별 수익률
- [ ] 위험 지표: 변동성, MDD, VaR (95%, 99%)
- [ ] 위험조정수익: Sharpe Ratio, Sortino Ratio, Calmar Ratio
- [ ] 거래 지표: 승률, 손익비, 평균 보유기간
- [ ] 벤치마크 대비: Alpha, Beta, Information Ratio
- [ ] 통계 검정: t-test for returns

## Output Schema

```python
class BacktestMetrics(BaseModel):
    total_return: float          # 총 수익률
    cagr: float                  # 연평균 수익률
    volatility: float            # 연간 변동성
    sharpe_ratio: float          # 샤프 비율
    sortino_ratio: float         # 소르티노 비율
    max_drawdown: float          # 최대 낙폭
    win_rate: float              # 승률
    profit_loss_ratio: float     # 손익비
    alpha: float                 # 알파
    beta: float                  # 베타
```
