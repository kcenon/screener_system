## Description

파워 유저를 위한 키보드 단축키

## Shortcuts

```
/       - 종목 검색 포커스
g + h   - 홈으로 이동
g + s   - 스크리너로 이동
g + p   - 포트폴리오로 이동
g + w   - 워치리스트로 이동
j / k   - 리스트 위/아래 이동
Enter   - 선택한 항목 열기
Esc     - 모달/검색 닫기
?       - 단축키 도움말
```
