## Core Features

### Stock Screening

Screen 2,400+ Korean stocks using 200+ indicators:

```markdown
Example: Find value stocks
- Market: KOSPI
- PER < 10
- PBR < 1.0
- ROE > 15%
- Debt Ratio < 100%
- Market Cap > 100B KRW
```

[Full Screening Guide →](/docs/guides/screening)

### Portfolio Management

Track your holdings and analyze performance:
- Add/remove holdings
- Track purchase prices and quantities
- Compare performance vs KOSPI/KOSDAQ
- View profit/loss and returns
- Export portfolio reports

[Portfolio Guide →](/docs/guides/portfolio)

### Price Alerts

Get notified when stocks meet your criteria:
- Price above/below threshold
- Price change percentage
- Volume spike detection
- Custom indicator conditions
- Multi-condition alerts

[Alerts Guide →](/docs/guides/alerts)

### Technical Analysis

Visualize price movements with:
- Candlestick charts
- Volume bars
- Moving averages (MA5, MA20, MA60, MA120)
- RSI (Relative Strength Index)
- MACD (Moving Average Convergence Divergence)
- Bollinger Bands
- Support/Resistance levels

[Chart Guide →](/docs/guides/charts)

### Financial Analysis

- Financial ratios (PER, PBR, ROE, ROA, etc.)
- Trend analysis
- Peer comparison

## Tips & Tricks

### Efficient Screening
1. Start with broad criteria, then narrow down
2. Use presets to save common screens
3. Combine fundamental and technical filters
4. Sort results by multiple criteria
5. Export results for further analysis

### Portfolio Optimization
1. Diversify across sectors and market caps
2. Set realistic profit/loss targets
3. Use alerts to monitor positions
4. Review portfolio performance regularly
5. Rebalance based on analysis

### Market Research
1. Follow top-performing stocks
2. Monitor insider trading activity
3. Track institutional buying/selling
4. Analyze sector trends
5. Stay updated with company news
